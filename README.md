# Plant Pest Classifier

A web application that identifies plant pests from images using an AI-based image classification model. Built with **React**, **TypeScript**, **Vite**, and styled with **Tailwind CSS**.

---

## Features

- Upload or take pictures of plants.
- AI-powered pest detection and classification.
- User-friendly interface with dynamic page navigation.
- Responsive design for desktop and mobile.

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **AI Model:** Pre-trained image classifier (TensorFlow / PyTorch can be integrated)
- **State Management:** React Hooks
